# Visual style презентации для руководства

## Назначение

Документ задает стиль management presentation по AI Discovery Platform. Он используется как brief для PowerPoint и Figma Slides.

## Общий стиль

- Enterprise / consulting / product strategy.
- Чистые схемы, крупные тезисы, минимум текста.
- Акцент на value, risk control, roadmap, architecture confidence.
- Русские заголовки и понятная терминология без перегруза техническим жаргоном.

## Светлая версия

- Фон: белый или очень светлый серо-синий.
- Основной текст: темный графит.
- Акцент: синий для platform/control, зеленый для readiness/value, желтый для risks.
- Использовать тонкие линии и простые блок-схемы.

## Темная версия

- Фон: глубокий графит/темно-синий.
- Текст: белый/светло-серый.
- Акцент: синий/циан для архитектуры, зеленый для value.
- Использовать умеренно, для титульного/разделительных слайдов.

## Слайды

- Один key message на слайд.
- До 3-5 bullets.
- Схемы должны объяснять процесс или архитектуру, а не украшать.
- Roadmap показывать как управленческий план 6-8 недель.
- Риски показывать вместе с controls.

## Не использовать

- Перегруженные таблицы на executive slides.
- Сырые backend class names без объяснения.
- Слишком много acronyms без расшифровки.
- Marketing hero вместо substance.
