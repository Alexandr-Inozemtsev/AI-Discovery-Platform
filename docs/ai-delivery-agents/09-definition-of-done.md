# Общий Definition of Done для Codex-задач

Задача считается завершенной, если выполнены все применимые условия:

- Обновлен код или документ, связанный с задачей.
- Не смешаны Product AI Agents и Global Codex Delivery Agents.
- Production-код runtime-агентов продукта не изменен без прямой необходимости и явного scope.
- Обновлен backlog, если изменился scope.
- Обновлен Trello package, если задача delivery-level.
- Обновлен Gantt, если меняются сроки, этапы или зависимости.
- Добавлены или обновлены тесты, если менялся код.
- Обновлена документация, если менялось поведение, API, UX, эксплуатация или release scope.
- Описаны риски и ограничения.
- Описан rollback, если есть production impact.
- Пройдены применимые quality gates.
- Итоговый отчет написан на русском языке.
